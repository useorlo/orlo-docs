# Migration Path to Platform

Many teams start by exploring Open Core and later adopt the Platform.

## Typical path

1. try validation or adapter packages
2. evaluate the Studio components
3. integrate the agent SDK against a live Platform endpoint
4. move task, dataset, and deployment workflows onto Orlo Platform

Open Core is not a dead end. It is a low-friction way to understand and adopt the Orlo model before committing to the full platform.
