# `@orlo/agent-sdk`

`@orlo/agent-sdk` is the developer-facing adoption layer for Orlo agent governance.

## Includes

- typed client
- session helpers
- semantic step helpers
- generic runtime wrappers

## Important note

This package talks to Orlo Platform. It is open-core code, but it expects a running governance backend.
