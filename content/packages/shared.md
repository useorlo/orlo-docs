# `@orlo/shared`

Use `@orlo/shared` for Orlo’s public contracts and reusable types.

## Includes

- runtime adapter interfaces
- validator rule interfaces
- scoring plugin interfaces
- execution interface contracts
- agent-governance types
- chunking helpers

## Does not include

- queue names
- platform defaults
- platform constants
- persistence logic
