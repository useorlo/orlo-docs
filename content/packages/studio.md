# `@orlo/studio`

`@orlo/studio` is a Web Components UI package for Orlo-native interfaces.

## Best fit today

- team tools
- demos
- Orlo-native dashboards
- custom portals that already speak the Orlo API contract

## Important note

It is reusable, but many components assume Orlo Platform endpoints and org-scoped headers.
